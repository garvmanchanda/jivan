# Assets Folder

This folder contains app icons and splash screens.

## Required Files

- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `splash.png` - Splash screen (1284x2778px)

## Placeholder Notice

The app will work without these files. Expo will use default placeholders.

To customize:
1. Create your icons using Figma/Sketch/Photoshop
2. Export at required sizes
3. Replace files in this folder
4. Run `npx expo prebuild --clean`
